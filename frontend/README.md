# NinaMart Management System

A modern retail management dashboard built with React, TypeScript, Tailwind CSS, and Recharts.

## Features

- **Dashboard** — KPI cards, sales charts, inventory tracking, staff performance
- **Sidebar Navigation** — Product, Sales, Staff, Reports, and Settings modules
- **Responsive Layout** — Adapts to desktop and mobile screens
- **Modern UI** — Clean design matching the NinaMart brand

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- Recharts (charts)
- Lucide React (icons)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
